Homefindify

Homefindify is an online tool for users looking for homes for sale or for rent. I was responsible for defining the look and feel of the brand as well as website design and front-end developemnt.

Branding

Influenced by user research, branding efforts included logo design, defining a color palette, choosing imagery, and developing a voice and tone.

User Flows

We wanted to create a user experience that served user expectations, finding home listings, and always feel like they were able to browse listings without having to constantly navigate backwards to be able to search.

Front-end Development

Working in a Ruby on Rails platform, I was responsible for the majority of the front-end development of html/css/js. Making my own commits and working closely with a developer we were able to build and maintain a highly functioning website at a rapid pace.

A/B Testing

As we were looking to acquire sign ups before granting a user full access to the full site, I led rigorous A/B testing to find the best experience that encouraged sign-ups, built trust, and managed user expectations.
